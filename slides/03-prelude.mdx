```PowerShell

$SanDiego = [Coordinate]@{ latitude = 32.7157; longitude = -117.1611 }

```

```PowerShell
# Fit a simple linear model

$X0 = 1,1,1,1,1 | matrix 5,1
$X1 = -2,-1,0,1,2 | matrix 5,1
$X = $X0.Augment($X1)
```
