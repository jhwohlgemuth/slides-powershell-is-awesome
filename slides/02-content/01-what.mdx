# What is PowerShell?

> ...a cross-platform task automation and configuration management framework, consisting of a command-line shell and scripting language.

## Examples

```PowerShell
$credential = Get-ChildItem Cert:\CurrentUser\My | ? { $_.FriendlyName -like "Digital Signature" }
```

```PowerShell
Get-ItemProperty HKLM:\SOFTWARE\WOW6432Node\Microsoft\Windows\CurrentVersion\Uninstall\* |
    Select-Object DisplayName, DisplayVersion, Publisher, InstallDate |
    Sort-Object DisplayName |
    Where-Object { $_.DisplayName.Length -gt 0 } |
    Format-Table -AutoSize
```

```PowerShell
Install-Module -Name Prelude -Scope CurrentUser
```
