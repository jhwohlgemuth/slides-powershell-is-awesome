# What is PowerShell?

> ...a cross-platform task automation and configuration management framework, consisting of a command-line shell and scripting language.
 [PowerShell Documentation](https://docs.microsoft.com/en-us/powershell/scripting/overview?view=powershell-7.1)

 ---
 
## In other words, PowerShell is... 
> A typed programming language focused on the user and installed on Windows by default

---

## Example
```powershell
$credential = Get-ChildItem Cert:\CurrentUser\My |
    ? { $_.FriendlyName -like "Digital Signature"  }
```
